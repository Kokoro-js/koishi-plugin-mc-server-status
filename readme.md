## 指令
> []: 可选参数 <>: 必选参数  
> 所有指令都不区分大小写

- `mcs [ip]` 查询Minecraft服务器状态，空则为默认IP或者群绑定IP
- `mcsBind <ip> [群号]` 绑定群内默认Minecraft服务器IP
